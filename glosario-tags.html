-Glosario de HTML-

body (block)- representa el contenido del documento de HTML Sólo puede haber un body en todo el documento

header (block) - representa contenido introducción de la pagina web puede contener logo, formulario de busqueda, nombre de autor, etc

hr (block) - Es una lina divisoria entre bloques

main (block) - representa el contenido principal del body de un documento. el contenido es relacionado directamente del tema principal del documento.

HTML es un lenguaje de marcado de texto que se usa para el desarrollo de páginas web, mediante el cual se crea la estructura y el formato de la información que se muestra al usuario encerrando esta información enetiquetas

--Semantica a nivel de texto--

a (inline) - Un Anchor Tag representa un hipervinculo a otro recurso, pagina web, url o sección del documento. puede abrirse en: _self: en la propia página _blank: nueva pestaña _parent: en la propia pagina _top: en la propia pagina

Ir a Google Biografía Ir a footer

abbr (inline) - Representa una abrebiación o acronimo, donde el atributo title contiene la descripcion del contenido abrebiado

Estoy aprendiendo HTML y pronto aprenderé hojas de estilo con CSS

b (inline) - Bring Atention to element o Dar atención, es usado para llevar al lector a un punto de atención, sirve para remarcar la atención dentro del texto marcandolo en negrita (bold)

strong (inline) - Se utiliza para marcar partes sumamente importantes, serias o de urgencia en el texto

Este texto es muy importante!

em (inline) - Se utiliza para marcar énfasis en partes importantes de un texto

El dinero es importante pero la salud lo es más

i (inline) - Texto Idiomatico, grosso modo representa una parte del texto que esta fuera del texto normal, ya sea una expresion idiomática, termino técnico, o menclaturas

dfn (inline) - La etiqueta de Definicion se usa para indicar un termino a definir dentro de un bloque de texto

br (inline) - Produce un salto de linea, se utiliza donde es importante los saltos como un poema o dirección

Las rosas son rojas
Las violetas azules
No soy bueno en poesia :(

s (inline) - Muestra el texto tachado (Strikethrough), para representar cosas que no tengan relevancia

Dentro de los textos el texto tachado tiene menor importancia

q (inline) - Indica una cita (Quote) corta dentro de un texto, encerrandolos en comillas dobles, para citas largas se necesita usar blockquote

Entonces Darth Vader le dijo al mal herido Luke: No Luke, yo soy tu padre

cite (inline) - Citacion, se usa para describir una referencia a una cita de un trabajo, un libro, una investigación, ensayo, una cancion, una pagina web

Para mas información consulte MDN web docs

kbd (inline) - Keyboard Input o Entrada de Teclado marca el texto que debe introducir el usuario

Si esta de acuerdo con los terminos y condiciones presione Aceptar

mark (inline) - Representa un texto remarcado como marcatexto para referencias o notas, el texto remarcado es de alguna relevancia o importancia en el texto

samp (inline) - Sample o ejemplo, se usa para representar texto como salida de un programa de computadora.

Mi computadora me decía:
Boot Error

bdo (inline) - Bidireccional Text Override, sobreescribe la direccion del texto y la cambia hacia otra direccion ltr de izquierda a derecha y rtl de derecha a izquierda

Este es un texto normal
Este es un texto alrevez

small (inline) - representa un texto pequeño, usado para copyright o una declaración legal

Este contenido esta protegido por Copyright 2020

ruby (inline) - Se utiliza para hacer anotaciones de texto arriba de un texto base, muy util en pronunciacion y traducción de caracteres asiaticos

rp (inline) - Ruby Fallback Parentesis, provee de soporte mediante parentesis para buscadores que no soportan las anotaciones ruby

rt (inline) - Ruby Text, especifica el contenido en la anotación de ruby

Texto base (Anotación)

明日 (Ashita) 明日 Ashita

rtc (inline) - Representa el conjunto de varios rb para formar una frase

馬(mǎ) 來(lái) 西(xī) 亞(yà) (Malaysia)

sub (inline) - Subscript, especifica un texto que se muestra como subindice

El atomo de agua se forma por H2O

sup (inline) - Superscript, muestra el texto como un super indice

La ecuación mas famosa es la de E=mc2

time (inline) - Representa un periodo específico de tiempo

El sol se ocultara a las 19:35 horas del 23 de Julio

span (inline) - Contenedor generico sin significado semantico, se usa con propositos de dar estilos a un bloque de texto o elementos mediante class y style

Este es un texto en un parrafo donde el span es este bloque de texto para cambiar estilos

var (inline) - Indica el nombre de una variable matematica o de programación

Para sacar el area de un cuadrado utilizamos a = l x l donde l es el tamaño del lado

wbr (inline) - Representa un salto de linea en una palabra (word break), donde el buscador puede opcionalmente hacer un salto en esa parte

Yo sé muy bien que la palabra favorita de Mary Poppins es Supercalifragilisticoespiralidoso

code (inline) - Muestra el texto indicando el estilo de un fragmento de código.

Este es un hola mundo en Java: System.out.println("Hola mundo");
